# Function Calling Benchmarks

## Prerequisites

Для загрузки бенчмарков необходимы `git`, `python3.10`, `uv`.

## Configuring and running
Перед загрузкой бенчмарков необходимо экспортировать переменную `BASE_URL`,
записав в нее адрес для обращений к API моделей. Для загрузки бенчмарков
используется `make` (`make prepare`). Запустить оценку на бенчмарках можно при
помощи `make <model_name>`, или `make eval_all` для оценки всех моделей,
предварительно добавив ключ в `.env`.
